About

This Sentiment analysis tool will predict the sentiment of the given input text. The systems used are listed below.

  • BoW This system uses a simple bag of words feature vector using support vector machines as a classifier.
  • LDA Latent Dirichlet allocation model is trained using two topics, positive and negative. The model is trained using a prior list of positive and negative words.
  • TNG The topical n-grams model is trained similarly using just two topics, positive and negative. The training phase makes use of a list of positive and negative words as well as rules to assign topics to bigrams.
  • UNL This is a rule based system. The input text is given to a UNL generator after pre-processing. The UNL graph thus generated is subjected to rules which calculate the sentiment. This is a ternary classifier. All the other systems are binary classifiers. It is not able to handle unstructured data.
  • Consensus The consensus algorithm is a simple summation algorithm which combines the outputs of BoW, Topical n-gram, and the UNL rule based classifier. The idea behind using these systems is that they belong to three different classes. BoW is supervised, Topical n-gram is semi-supervised, and BoW is completely unsupervised. Here, more weightage is given to the UNL classifier because it performs the best for structured data. The reason to include the other two systems is to handle unstructured data as well. UNL system usually gives objective output for unstructured data as it is not able to parse it properly.
